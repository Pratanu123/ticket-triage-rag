# Getting Started with CloudLedger

CloudLedger is a SaaS platform for expense tracking, invoicing, and team reimbursements.

## Create your workspace
1. Sign up at https://app.cloudledger.example with a work email.
2. Verify your email, then create a workspace name (this becomes your subdomain).
3. Invite teammates from **Settings → Team → Invite**.

## First invoice
1. Go to **Invoices → New Invoice**.
2. Add a customer, line items, tax rate, and due date.
3. Click **Send** to email a payment link, or **Download PDF**.

## Mobile access
CloudLedger works in modern browsers on desktop and mobile. There is no separate
native app yet; use the Progressive Web App install prompt if offered.
