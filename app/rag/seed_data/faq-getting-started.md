# FAQ: Getting Started

## What is CloudNova?
CloudNova is a SaaS workspace for project tracking, task boards, and light
resource planning for product and ops teams.

## Create a workspace
1. Sign up at https://app.cloudnova.example with a work email
2. Verify email, choose a workspace slug
3. Invite teammates from **Settings → Team → Invite**

## First project
**Projects → New project →** pick a template (Kanban, Sprint, or Blank) → add
tasks. Starter includes 3 projects; Pro is unlimited.

## Mobile
Use the mobile web app; a native iOS/Android app is on the roadmap. Offline
drafts cache in the browser for up to 7 days.
