# Login: Password Reset

## Reset a forgotten password
1. Visit https://app.cloudnova.example/login
2. Click **Forgot password**
3. Enter the email on the account
4. Open the reset link within **30 minutes**

If the email does not arrive, check spam and confirm you are using the same
address as your CloudNova account. Corporate filters sometimes quarantine
`noreply@cloudnova.example`.

## After reset
You are signed out of other sessions. If you use SSO, password reset only
applies when password login is still enabled for your workspace.
